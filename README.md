# python-google-drive
Pythonを使ってGoogle Driveと情報のやり取りをしよう。

## 必要なもの
- https://console.developers.google.com/apis/credentials

```
pip install --upgrade google-api-python-client google-auth-httplib2 google-auth-oauthlib
```
